Medicine Reminder — responsive UI
Files:
 - index.html
 - styles.css
 - app.js

How to use:
Open index.html in a browser. This is a front-end only demo that stores medication data in localStorage.
Features:
 - Week calendar (select a day)
 - Add/Edit/Delete medications for a selected day
 - Responsive: mobile-first and desktop layout
 - Modal for adding medicines
